# Delta: Web Frontend — Comment Amplify

## ADDED Requirements

### REQ-WEB-DIS-AMPLIFY-001

Discover comment cards and post comment actions SHALL show Author+-gated repost menu (repost / undo / quote), share deep-link, and amplify badge when `repost_count > 0`.

### REQ-WEB-DIS-AMPLIFY-002

Quote-from-comment compose SHALL create a published post with `quoted_comment_id` and preview the embedded comment.

### REQ-WEB-DIS-AMPLIFY-003 (Phase 2)

Discover page SHALL expose topic/interest filter chips parity with Android.
