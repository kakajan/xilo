# Delta: Android Native — Comment Amplify

## ADDED Requirements

### REQ-AND-DIS-AMPLIFY-001

`CommentCard` (Discover + threads) SHALL show Author+-gated `RepostMenuButton` for comments, share deep-link, and amplify badge when `repost_count > 0`.

### REQ-AND-DIS-AMPLIFY-002

`CreatePost` SHALL accept `quotedCommentId`, load comment summary, and submit `quoted_comment_id`.

### REQ-AND-DIS-AMPLIFY-003 (Phase 2)

Pin control for post author/mod on comments SHALL be available from thread UI.
